我是王鑫，运营 **天智工坊（Tianmind Studio）**。

我做的不是“看起来像 demo 的玩具项目”，而是面向真实使用场景的交付：
- 能上线的官网与落地页
- 能落地的后台 / SaaS / 内部工具
- 能接入业务流的 AI 功能与 Agent 工作流
